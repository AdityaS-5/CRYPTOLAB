📝 RSA String Encryption

Encrypt text by converting each character to ASCII and encrypting individually

ℹ️ String Encryption Process:

  1. Convert each character to its ASCII value
  2. Encrypt each ASCII value using RSA: ci ≡ mie mod n
  3. Concatenate all encrypted values with spaces

Note: All ASCII values must be < n (typically n should be > 127 for standard ASCII)
